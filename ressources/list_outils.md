# Useful Tools for RootMe Challenges

## Web Challenges
- **Burp Suite** - Web vulnerability scanner and proxy
- **OWASP ZAP** - Web application security scanner
- **SQLMap** - Automatic SQL injection tool
- **Dirb/Dirbuster** - Directory brute-forcing

## Cryptography
- **CyberChef** - Web-based encryption/encoding tool
- **John the Ripper** - Password cracking
- **Hashcat** - Advanced password recovery

## Forensics
- **Wireshark** - Network protocol analyzer
- **Binwalk** - Firmware analysis tool
- **Steghide** - Steganography tool
- **Exiftool** - Metadata viewer

## Reverse Engineering
- **Ghidra** - Software reverse engineering suite
- **Radare2** - Unix-like reverse engineering framework
- **IDA Pro** - Interactive disassembler
- **Binary Ninja** - Reverse engineering platform

## System
- **Nmap** - Network discovery and security auditing
- **Metasploit** - Penetration testing framework
- **Netcat** - Networking utility

## Programming
- **Python** - General purpose scripting
- **Pwntools** - CTF framework for Python
- **Ruby** - Useful for some exploits
