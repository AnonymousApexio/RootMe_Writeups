## Steganography Tools

### General Image Analysis
- **StegSolve** - Advanced image viewer for LSB analysis
- **StegOnline** - Web-based version of StegSolve
- **GIMP** - Advanced image manipulation
- **ImageMagick** - Command-line image processing
- **ExifTool** - Metadata extraction and manipulation
- **Exiv2** - Image metadata manipulation

### PNG-specific Tools
- **pngcheck** - Verify PNG integrity and extract chunks
- **zsteg** - Detect stegano-hidden data in PNG & BMP
- **stegpy** - Python tool for LSB steganography
- **stegano** - Python steganography toolkit
- **stegoveritas** - Automated steganography detection

### JPEG-specific Tools
- **steghide** - Hide data in JPEG files
- **jsteg** - Traditional JPEG steganography tool
- **outguess** - Universal steganography tool
- **jphide/jpseek** - JPEG steganography suite
