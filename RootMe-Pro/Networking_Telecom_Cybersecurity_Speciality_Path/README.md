# Course Description:
This course encompasses all the modules and challenges associated with the BUT (Bachelor of Technology). Through this course, participants will have the opportunity to explore various introductory topics related to cybersecurity.

The following themes will be covered in this course:

- Introduction to web vulnerabilities (BLE, APIs, etc.);
- Network capture analysis (PCAP analysis and attack identification);
- Log analysis (Web, Linux, Active Directory, etc.);
- Advanced web vulnerabilities (file upload, BLE, SQLi);
-Privilege escalation on Linux (SUID, sudo, crontab, etc.);

Through this course, participants will acquire the fundamental technical skills necessary to work on more advanced topics.
